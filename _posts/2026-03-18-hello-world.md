## 欢迎来到我的新博客！🎉

这篇博客是基于 **Jekyll** 和 **GitHub Pages** 搭建的。它不仅不需要服务器，还能直接通过 Git 推送本地 Markdown 文件来发布文章。非常方便、极客且免费！

### 为什么选择这个方案？

传统博客往往受限于固定的主题或臃肿的后台管理系统，而这个方案带来了以下好处：

1. **零成本部署**：完全依托于 GitHub Pages 强大的基础设施。
2. **本地编辑**：使用你最喜欢的 Markdown 编辑器（比如 VS Code 或 Typora）在本地写文章。
3. **极简设计**：采用了深蓝色调（参考了明日方舟的科技感风格），让读者能够专注内容本身。

### Markdown 支持演示

这里可以非常优雅地展示代码：

```javascript
// 这是一个简单的 JavaScript 示例
function sayHello(name) {
  console.log(`Hello, ${name}! Welcome to my blog.`);
}

sayHello('Doctor');
```

当然，也支持各种列表和引用：

> "The best time to plant a tree was 20 years ago. The second best time is now."

- 支持分类和标签系统
- 支持响应式布局
- 支持平滑的动画效果
